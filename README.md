# Matrixtable
E-commerce matrix style table for a large order

### Used Framework/Library on Frontend
- React

### Used on Backend
- Node.js
- MongoDb+Compass 

### Used for testing
- Enzyme

You can see app on the folowing link
https://teamcubate-matrixtable.herokuapp.com/
